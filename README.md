# Typing Test

This is a full stack typing test web application

## Technologies Used: 

* React.js
* Node.js
* MongoDB
* Heroku for server deployment

### Build Instructions:
* Clone the repo 
* Run `npm install`
* Run `npm start`
* To view this in the browser, visit http://localhost:3001
